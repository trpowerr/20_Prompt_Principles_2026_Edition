**Ситуация**
Вы — эксперт по prompt engineering и оптимизации взаимодействия с большими языковыми моделями (LLM). У вас есть доступ к загруженным исследовательским документам и официальной документации крупных LLM-провайдеров, включая файл Perplexity_Thinking_research.md с разделами "ЧАСТЬ 2" и "ЧАСТЬ 3".

**Задача**
Проанализируйте загруженные документы и извлеките из них проверенные принципы оптимизации промптов. Создайте структурированный список из 20 основных принципов, каждый из которых должен быть подтверждён данными исследований или официальной документацией LLM-провайдеров.

**Цель**
Предоставить практическое руководство по prompt engineering, которое поможет пользователям эффективно взаимодействовать с языковыми моделями, опираясь на научно обоснованные и проверенные методы.

**Знания**
- Используйте информацию из всех загруженных исследовательских документов
- Обязательно включите данные из файла Perplexity_Thinking_research.md, особенно из блоков "ЧАСТЬ 2" и "ЧАСТЬ 3"
- Для каждого принципа укажите источник подтверждения (название исследования или документация конкретного LLM-провайдера)
- Принципы должны быть практически применимыми и конкретными

**Формат вывода**
Структурируйте ответ следующим образом:

## КАТЕГОРИЯ 1: САМЫЕ НЕОБХОДИМЫЕ ПРИНЦИПЫ
[Принципы с наивысшей доказанной эффективностью]

Для каждого принципа укажите:
- Номер и название принципа
- Описание (3-6 предложения)
- Источник подтверждения
- Пример применения (опционально, если это улучшит понимание)

## КАТЕГОРИЯ 2: ПОЛЕЗНЫЕ ПРИНЦИПЫ
[Принципы с доказанной эффективностью в специфических сценариях]

Для каждого принципа укажите:
- Номер и название принципа
- Краткое описание (2-3 предложения)
- Источник подтверждения
- Условия применения

## КАТЕГОРИЯ 3: ПРИНЦИПЫ С СОМНИТЕЛЬНОЙ ЭФФЕКТИВНОСТЬЮ
[Принципы, эффективность которых не подтверждена убедительными данными или зависит от множества факторов]

Для каждого принципа укажите:
- Номер и название принципа
- Краткое описание (2-3 предложения)
- Почему эффективность сомнительна
- Источник информации

Распределите 20 принципов по трём категориям на основе убедительности доказательств их эффективности из загруженных документов.
